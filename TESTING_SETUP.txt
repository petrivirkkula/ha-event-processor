1. INSTALL TEST DEPENDENCIES
   $ pip install -r requirements-test.txt

2. RUN TESTS (Basic)
   $ pytest tests/

3. RUN TESTS WITH COVERAGE
   $ pytest tests/ --cov=src/ha-event-processor

4. GENERATE HTML COVERAGE REPORT
   $ pytest tests/ --cov=src/ha-event-processor --cov-report=html
   # Open htmlcov/index.html in browser

5. RUN AUTOMATED TEST SCRIPT
   $ bash scripts/run-tests.sh

================================================================================
                          TEST COMMANDS
================================================================================

# Run all tests with verbose output
pytest tests/ -v

# Run specific test file
pytest tests/test_config.py

# Run specific test
pytest tests/test_config.py::TestSettings::test_default_values

# Run tests matching pattern
pytest tests/ -k "mqtt"

# Run with coverage and show missing lines
pytest tests/ --cov=src/ha-event-processor --cov-report=term-missing

# Generate all reports (HTML, XML, JUnit)
bash scripts/run-tests.sh

# Run failed tests only
pytest tests/ --lf

# Run with debug output
pytest tests/ -s

# Run with long traceback
pytest tests/ --tb=long

Coverage reports are generated in multiple formats:

HTML Report:
  Location: htmlcov/index.html

  Interactive source browsing

XML Report:
  Location: coverage.xml
  Compatible with SonarQube, Codecov
  Machine-readable format

JUnit XML Report:
  Location: test-results.xml
  Compatible with CI/CD systems
  Jenkins, GitLab CI, GitHub Actions

Terminal Report:
  Shows coverage percentage per module
  Lists missing lines (with --cov-report=term-missing)
